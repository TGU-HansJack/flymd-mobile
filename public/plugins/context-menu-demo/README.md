# 右键菜单演示扩展

这是一个演示 flyMD 右键菜单 API 的示例扩展，展示了 `addContextMenuItem` API 的各种用法。

## 功能特性

### 1. 文本转换
- 转换为大写（快捷右键菜单项）
- 转大写 / 转小写 / 首字母大写
- 驼峰命名 (camelCase)
- 蛇形命名 (snake_case)
- 短横线命名 (kebab-case)

### 2. 空格处理
- 去除所有空格
- 压缩多余空格

### 3. 格式化工具
- 格式化 JSON
- 压缩 JSON
- URL 编码/解码
- Base64 编码/解码

### 4. 插入工具
- 当前日期时间
- ISO 时间戳
- Unix 时间戳
- UUID
- 随机字符串

### 5. 信息查看
- 字符数统计
- 单词数统计
- 行数统计
- 当前源码模式
- 文件名显示

## 安装方式

### 方式一：本地安装（已内置）
本扩展已内置在 flyMD 的 `public/plugins/context-menu-demo` 目录中，可直接在扩展管理中启用。

### 方式二：从源码安装
如果您想自己修改或学习，可以：
1. 将 `context-menu-demo` 文件夹复制到 flyMD 的插件目录
2. 在扩展管理中刷新并启用

## 使用方法

1. 在 flyMD 的扩展管理中启用"右键菜单演示"
2. 在编辑器中选中文本
3. 右键点击查看可用的菜单选项
4. 根据需要选择相应的功能

**提示：**
- 悬停在带箭头（▸）的菜单项上可展开子菜单
- 按住 `Shift` 键再右键点击可显示浏览器原生菜单

## 特色功能展示

### 条件显示
部分菜单项只在满足条件时显示：
- 文本转换菜单仅在有选中文本时显示
- 插入功能无需选中文本即可使用

### 子菜单
所有功能都按类别组织成子菜单，方便查找和使用。

### 上下文感知
菜单项可以访问当前的编辑上下文，包括：
- 选中的文本
- 光标位置
- 当前模式（源码/预览/所见）
- 当前文件路径

## 开发参考

本扩展是学习 flyMD 右键菜单 API 的绝佳示例，涵盖了：
- 简单菜单项
- 带子菜单的菜单项
- 条件显示
- 分组和分隔线
- 上下文对象的使用
- 文本替换和插入

详细的 API 文档请参考 [plugin.md](../../../plugin.md)。
